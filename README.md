# alimama-webapi
无需阿里开放平台淘客权限，实现服务端全自动淘客抓取、佣金计划申请、优惠券生成、淘客链接创建等功能

# 目标

完全独立运行与服务器端的淘客数据生成工具，实现：

- 商品数据采集
- <b>采用非淘宝开放平台api</b>
- 查询是否加入淘宝客
- 申请淘客最高佣金计划
- 创建商品、店铺优惠券
- 创建淘客连接

# 思路

## 商品数据采集
商品数据采集主要有两种方式：

- 采集市面上比较大的淘客平台发布站点，如：实惠猪、淘客助手、大淘客、好单库等等网站
- 直接去阿里妈妈后台的超级搜索页面采集
- 其他方式...
